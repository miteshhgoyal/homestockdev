# HomeStock Desktop App

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

**HomeStock** is a desktop automation tool for downloading and processing stock market data from NSE (National Stock Exchange) and BSE (Bombay Stock Exchange).

---

##  Features

**Automated Downloads** - Download NSE Bhavcopy, NSE Delivery data, and BSE Bhavcopy files  
**Excel Processing** - Transform CSV files into formatted Excel reports  
**Daily Scheduler** - Set automated daily downloads at your preferred time  
**File Management** - View, download, and delete files from the app  
**Real-time Logs** - Monitor all operations with detailed logging  
**Cross-Platform** - Works on Windows and macOS
